## Створення чат-боту «Прогноз погоди» на Python для Telegram.  

### Додаткові матеріали:  

### Вимоги:  

  [Робота з GIT](https://itvdn.com/ru/webinars/description/git-practicum)  
  [Створення чат-бота «Прогноз погоди» на Python](https://www.youtube.com/watch?v=JekOB5JjuMI)


* Вихідний код має бути розміщений на github або іншому подібному сервісі.  

* Додайте роботу з Telegram API. Деталі можна переглянути за посиланням:
[https://api.telegram.org](https://api.telegram.org)  

* Під'єднайте бібліотеки до Telegram API. 

* Для отримання даних із сервера використовуйте дані сайту 
[https://openweathermap.org](https://openweathermap.org) у такому вигляді: [http://api.openweathermap.org/data/2.5/weather?q={city}&appid={token}](http://api.openweathermap.org/data/2.5/weather?q={city}&appid={token}).
